EMBED PROPERTIES USED:
─────────────────────────────────────────────────────────────────────
• setTitle()        - Command title/heading
• setDescription()  - Main content area
• setFooter()       - Attribution and info
• setThumbnail()    - Small image (avatars/icons)
• setColor()        - Border color
• setTimestamp()    - Automatic timestamp

NOT USED (as requested):
─────────────────────────────────────────────────────────────────────
✗ Fields (addFields)      - Not used
✗ Author (setAuthor)      - Not used
✗ Image (setImage)        - Not used
✗ Emojis                  - Not used anywhere
